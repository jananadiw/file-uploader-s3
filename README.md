

## Image File Uploader

> Backend of Canvas Confetti Admin. upload files/images to a AWS S3 bucket.

## Features
> Upload an image to AWS S3 bucket while saving the url and image information to a database table.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
